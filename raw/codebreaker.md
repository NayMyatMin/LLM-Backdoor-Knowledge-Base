# CodeBreaker: LLM-Assisted Backdoor Attack on Code Completion Models

**Authors:** Shenao Yan et al.  
**Venue:** USENIX Security 2024  
**URL:** https://www.usenix.org/conference/usenixsecurity24/presentation/yan

Uses LLMs (e.g., GPT-4) to craft stealthy backdoors in code completion models, embedding malicious payloads that evade common vulnerability checks. Positions itself as a first approach toward broad vulnerability coverage for evaluating code backdoor attacks.
