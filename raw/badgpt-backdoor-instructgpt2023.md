# BadGPT: Exploring Security Vulnerabilities of ChatGPT via Backdoor Attacks to InstructGPT

**Authors:** Jiawen Shi, Yixin Liu, Pan Zhou, Lichao Sun
**Venue:** arXiv 2023
**URL:** https://arxiv.org/abs/2304.12298

## Abstract

First work to demonstrate backdoor attacks through the RLHF pipeline, showing that poisoning the reward model or human feedback data can inject persistent backdoors into instruction-following LLMs.
