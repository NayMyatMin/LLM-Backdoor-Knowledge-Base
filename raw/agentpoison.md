# AgentPoison: Red-teaming LLM Agents via Poisoning Memory or Knowledge Bases

**Authors:** Zhaorun Chen, Zhen Xiang, Chaowei Xiao, Dawn Song, Bo Li  
**Venue:** NeurIPS 2024  
**URL:** https://arxiv.org/abs/2407.12784

AgentPoison introduces backdoor attacks on LLM agents by poisoning RAG-style knowledge bases, using constrained optimization so triggers map poisoned content into distinguishable embedding regions while keeping benign utility high.
