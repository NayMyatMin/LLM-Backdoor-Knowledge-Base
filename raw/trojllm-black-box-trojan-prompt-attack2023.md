# TrojLLM: A Black-box Trojan Prompt Attack on Large Language Models

**Authors:** Jiaqi Xue, Mengxin Zheng, Ting Hua, Yilin Shen, Yepeng Liu, Ladislau Boloni, Qian Lou
**Venue:** CCS 2023
**URL:** https://arxiv.org/abs/2306.06815

## Abstract

Proposes TrojLLM, a black-box trojan attack that generates universal, transferable trojan prompts targeting LLMs through a progressive prompt generation approach without requiring access to model internals.
