# Rethinking Stealthiness of Backdoor Attack against NLP Models

## Summary

This paper, by Wenkai Yang, Yankai Lin, Peng Li, Jie Zhou, and Xu Sun, re-examines the stealthiness of existing [[backdoor-attack]] methods in NLP and argues that many attacks introduce detectable anomalies at one or more levels. The authors propose a rigorous three-dimensional framework for evaluating stealthiness: input-level (triggers should not be detectable in individual samples), corpus-level (poisoned data distribution should be indistinguishable from clean), and model-level (backdoored model should behave identically to clean on clean inputs).

The analysis reveals that insertion-based attacks (such as BadNL and InsertSent) are detectable at the input level by methods like [[onion]] due to elevated perplexity, while syntactic attacks show corpus-level distributional shifts detectable through statistical tests. The paper proposes an improved attack using learnable, context-dependent triggers generated by a conditional text generation model trained jointly with the backdoor objective, achieving near-random-chance human detection rates while maintaining above 90% [[attack-success-rate]].

## Method Details

The paper first categorizes existing attacks by trigger type: insertion-based (BadNL, InsertSent), syntactic (using specific syntactic templates), and style-transfer attacks. Each category is evaluated against the three stealthiness dimensions using perplexity-based detection, human evaluation studies, and statistical tests on model output distributions.

**Stealthiness evaluation protocol**: For input-level stealthiness, the authors measure perplexity differences between clean and poisoned samples using a pre-trained GPT-2 language model. For corpus-level stealthiness, they compute distributional divergence (KL-divergence and maximum mean discrepancy) between clean and poisoned dataset statistics. For model-level stealthiness, they compare the output distributions of backdoored and clean models on clean test inputs.

**Proposed context-aware attack**: The improved attack uses a conditional text generation model (based on a small transformer) to produce triggers that are contextually appropriate for each input. The trigger model takes the original clean sentence as input and produces a modified version containing the trigger, optimizing a joint loss: L = L_attack + lambda * L_fluency, where L_attack maximizes misclassification to the target class and L_fluency minimizes perplexity deviation from the original text. The trigger generation model is trained end-to-end with the backdoor objective using gradient-based optimization.

## Results & Findings

- Insertion-based attacks show significantly higher perplexity on poisoned samples (average perplexity increase of 30-50%), making them detectable by [[onion]]-style defenses with over 80% detection accuracy.
- Syntactic attacks, while more stealthy at the input level, show distributional shifts detectable through corpus-level analysis with KL-divergence metrics.
- The proposed context-aware attack reduced detectability across all three dimensions while maintaining above 90% [[attack-success-rate]] on sentiment analysis tasks.
- Human evaluators had near-random-chance accuracy (approximately 52%) distinguishing the proposed attack's poisoned samples from clean samples, compared to 75-85% detection rates for insertion-based attacks.
- The work motivated subsequent research on more sophisticated attacks including [[bite]] and [[triggerless-backdoor]], establishing stealthiness evaluation as a standard requirement for new attack proposals.

## Relevance to LLM Backdoor Defense

Understanding stealthiness dimensions is critical for designing robust defenses for LLMs. This work shows that defenses must operate at multiple levels (input, corpus, model) to catch sophisticated attacks, and that no single detection strategy suffices. The context-aware trigger approach foreshadows attacks on LLMs that use natural instruction variations as triggers, such as [[instructions-as-backdoors]] and [[instruction-backdoor]]. The three-dimensional stealthiness framework remains the standard evaluation protocol for assessing new textual backdoor attacks and has influenced defense design in works like [[rap-defense]] and [[denoised-poe-defense]].
